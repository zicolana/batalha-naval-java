# 🚢 Batalha Naval em Java

Jogo de batalha naval em console desenvolvido em Java.

## 💻 Tecnologias
- Java
- MySQL

## 🎮 Funcionalidades
- Jogo contra bot
- Sistema de ranking
- Validação de jogadas
- Interface em console com feedback visual

## ▶️ Como rodar
1. Clone o repositório:
   git clone https://github.com/SEU_USUARIO/NOME_DO_REPO.git

2. Configure o banco de dados no arquivo Ranking.java

3. Execute a classe Main
